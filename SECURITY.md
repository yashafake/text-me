# Security Policy

## Supported versions

Security fixes are applied to the latest `main` branch state.

## Reporting a vulnerability

Please do not open public issues for security vulnerabilities.

Use GitHub Security Advisories:
- [Report a vulnerability](https://github.com/yashafake/text-me/security/advisories/new)

Include:
- clear reproduction steps
- impact assessment
- affected platform/version

We will review reports as quickly as possible and coordinate disclosure.
