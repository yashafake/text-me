# Contributing

Thanks for your interest in improving TextMe.

## Development setup

1. Install Node.js 18+.
2. Install dependencies:

```bash
npm install
```

3. Run the app:

```bash
npm start
```

## Testing

Run unit tests:

```bash
npm test
```

Run e2e external-link smoke test:

```bash
npm run test:e2e
```

## Pull request guidelines

- Keep PRs focused and small.
- Include test evidence in PR description.
- Update docs if behavior or setup changed.
- Do not include secrets, local configs, build artifacts, or `node_modules`.
