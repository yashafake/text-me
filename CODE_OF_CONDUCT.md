# Code of Conduct

## Our standard

We are committed to a friendly, safe, and respectful community for everyone.

Examples of expected behavior:
- Be respectful and constructive.
- Assume good intent and collaborate in good faith.
- Give and receive feedback professionally.

Examples of unacceptable behavior:
- Harassment, discrimination, or hate speech.
- Personal attacks, insults, or trolling.
- Publishing private information without consent.

## Enforcement

Project maintainers are responsible for clarifying and enforcing this Code of Conduct and may take corrective action for behavior they deem inappropriate, threatening, or harmful.

## Scope

This Code of Conduct applies in project spaces, including issues, pull requests, discussions, and related communication channels.

## Reporting

Report unacceptable behavior to project maintainers via GitHub private channels where possible.
